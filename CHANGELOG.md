# Changelog

## [1.1.0] - 2026-03-09

### Added
- **Unified Data Architecture**: Integrated behavioral signals (timing, patterns) with semantic narrative analysis.
- **UserPortrait Aggregation**: Automated consolidation of session results into a sphere-based consciousness imprint.
- **Context-Aware Agents**: `SyncAgent` and `AnalyticAgent` now retrieve historical context from `UserPortrait`.
- **System Documentation**: Comprehensive technical manual for the v1.1 architecture.
- **Maintenance Scripts**: Organized utility scripts into `scripts/maintenance/`.

### Changed
- **Text-Centric Shift**: Transitioned from visual diagnostics to purely textual narrative layers.
- **Project Structure**: Reorganized root directory, moving deployment and utility files to dedicated folders (`infra/`, `scripts/`, `tests/`).
- **Analytic Agent Refactor**: Updated the Mirror analysis algorithm with 4-stage processing.

### Removed
- Legacy visual diagnostic models and endpoints.
- Unused deployment scripts from the root directory.
